By this plugin you can Ban / Lock any user you want

== Description ==

With this plugin you can block any bad users without deleting the account and then any time you want you can unlock the user.

== Attributes ==

* Ban / Locked Bad User.
* Force Logout The User You Locked.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the users part and select the user that you wish ban and then edit that user and find Lock User Account part and select Locked or Unlocked option from Account status.
4. This will work for any other users you want.
